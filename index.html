DÉVELOPPEMENT WEB

Dates de création :

2018 - 2019


Contexte :

Projet « chef-d’oeuvre », réalisé sur 9 mois, dans le cadre du passage du titre professionnel Concepteur et Développeur d'Applications.


Client :

Projet personnel


Objectif du site :

Aider les personnes en dépression et isolées, en leur proposant un moyen d'écrire et d'organiser leurs pensées. En d’autres termes, permettre aux personnes en dépressions de panser leurs blessures émotionnelles en leur proposant d’organiser leurs pensées.


Technologies utilisées :

HTML - CSS - Javascript - MySql - Vue.js - Node.js - Méthodologie de design UX - Marvel app - Git - Github
